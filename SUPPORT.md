# Getting Help

## Questions

Use [GitHub Discussions](https://github.com/motebit/motebit/discussions) for questions about usage, architecture, or integration. We monitor discussions actively.

## Bug reports

File an [issue](https://github.com/motebit/motebit/issues/new?template=bug_report.yml) using the bug report template. Include steps to reproduce and your environment.

## Security vulnerabilities

**Do not file a public issue.** Email **security@motebit.com**. See [SECURITY.md](SECURITY.md) for the full policy.

## Documentation

- [Identity specification](spec/identity-v1.md) -- the open standard for agent identity files
- [Docs site](https://docs.motebit.com) -- guides for developers, operators, and app reference

The platform is [BSL 1.1](LICENSE) -- free for non-commercial use, source-available, converts to MIT after 4 years. For commercial licensing inquiries, email **daniel@motebit.com**.
